[Ambiente Web]
Sendo um usuário do site WebMotors
Quero fazer uma busca por um determinado modelo veículo 

Dado que acesso pagina inicial de busca por veiculos
Pesquiso por Honda City e seleciono o veiculo, sou redirecionado para uma página onde me permite me visualizar os anúncios do referido veículo e a seleção de filtros avançados 
Seleciono a versão do veículo desejado 
Escolho filtrar por de anuncios de loja 
Então, é mostrado na tela todos os anuncios referente ao veículo buscado, onde, os vendedores são somente "Lojas"

Dado que selecionei um anuncio
Quando seleciono a opção para ver todo o estoque daquela loja 
Entao é mostrado todos os anuncios de veículos daquela determinada loja

[API] 

Dado a requisição para verificar as marcas de veiculo 
Então é me retornado os ID das marcas
Dado o ID da Marca, é feita a consulta dos modelos de veículos
Dado o ID do modelo desejado, é feita a consulta pelas versões disponíveis
